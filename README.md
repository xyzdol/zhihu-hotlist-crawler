# Zhihu Hotlist Crawler

A simple Python crawler that fetches the Zhihu hot list through the official API and saves the result as JSON.

## Features
- Fetches real-time Zhihu hot list
- Extracts title, heat, and link for each item
- Automatically saves results with timestamped filenames
- Clean and easy-to-read JSON structure

## Usage

```bash
python main.py
